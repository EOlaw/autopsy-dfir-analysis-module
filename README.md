# Autopsy DFIR Analysis Module

A Python-based ingest module for Autopsy designed for Digital Forensics
and Incident Response (DFIR) investigations.

## Features
- File system analysis (NTFS / ext4)
- Deleted file detection
- Timeline artifact creation
- Keyword searching
- Executable identification
- Autopsy Blackboard integration

## Installation
1. Copy `Autopsy_DFIR_Module` into:
   Autopsy/python_modules/
2. Restart Autopsy
3. Enable "DFIR Analysis Module" during ingest

## Use Cases
- Insider threat investigations
- Data exfiltration cases
- Corporate policy violations
- Malware discovery
